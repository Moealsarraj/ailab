# AI Lab — AI Research & Evaluation Suite

Four tools for testing, benchmarking, and hardening AI systems. Built for researchers, AI engineers, and prompt designers.

| Tool | What it does |
|------|-------------|
| **Prompt Bench** | Benchmark prompts across models and compare outputs side-by-side |
| **Prompt Shield** | Test prompts for jailbreaks, injection attacks, and adversarial inputs |
| **Agent Builder** | Design and prototype multi-step AI agent workflows visually |
| **Arabic Bench** | Evaluate Arabic language understanding and generation quality across models |
